Словообразование

Темы по словообразованию:
  • Префиксы
  • Суффиксы
  • Составные слова

Префиксы

Определение: Префиксы добавляются в начало слов, чтобы изменить их значение. Они могут передавать различные значения, такие как отрицание, изменение, направление и т. д.

Примеры:

  • "un-" + "happy" = "unhappy" (несчастный)
  • "dis-" + "agree" = "disagree" (не согласен)
  • "pre-" + "view" = "preview" (предварительный просмотр)
  • "re-" + "write" = "rewrite" (переписать)
Упражнение: Заполните пропуски, добавляя префиксы к следующим словам:
  • happy -> _____ (несчастный)
  • appear -> _____ (не появляется)
  • cycle -> _____ (переключить)
  • charge -> _____ (перезарядить)

Суффиксы

Определение: Суффиксы добавляются к концу слов, чтобы образовать новые слова, часто изменяя их грамматическую категорию (например, из существительного в прилагательное).

Примеры:

  • "teach" + "-er" = "teacher" (учитель)
  • "care" + "-ful" = "careful" (внимательный)
  • "beauty" + "-ful" = "beautiful" (красивый)
  • "child" + "-ren" = "children" (дети)
Упражнение: Добавьте суффиксы к следующим словам, чтобы образовать новые:
  • joy -> _____ (радость)
  • play -> _____ (игрок)
  • friend -> _____ (дружелюбный)
  • care -> _____ (забота)

Составные слова

Определение: Составные слова образуются путем соединения двух или более слов, которые создают новое значение.

Примеры:

  • "note" + "book" = "notebook" (блокнот)
  • "sun" + "flower" = "sunflower" (подсолнечник)
  • "tooth" + "brush" = "toothbrush" (зубная щетка)
  • "mother" + "in-law" = "mother-in-law" (тёща)
Упражнение: Образуйте составные слова, соединяя следующие пары слов:
  • foot + ball -> _____ (футбол)
  • black + board -> _____ (доска)
  • rain + bow -> _____ (радуга)
  • swimming + pool -> _____ (бассейн)
