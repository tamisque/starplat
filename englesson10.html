Неопределённые местоимения Some/Any

Some и Any

Темы урока:

  • 1. Введение
  • 2. Когда использовать some и any
  • 3. Примеры использования
  • 4. Практика
  • 5. Домашнее задание

Введение:

  • Неопределённые местоимения **some** и **any** употребляются для обозначения количества предметов или существ.
  • Знание того, когда использовать каждое из них, помогает грамотно строить предложения.

Когда использовать some и any:

  • Some обычно употребляется в утвердительных предложениях и в вопросах, если ожидается положительный ответ. Пример: "I have some apples." (У меня есть несколько яблок.)
  • Any используется в вопросах и отрицательных предложениях, когда количество может быть нулевым. Примеры: "Do you have any apples?" (У вас есть яблоки?) или "I don’t have any apples." (У меня нет яблок.)

Примеры использования:

  • Утвердительное предложение с **some**: "She bought some flowers." (Она купила несколько цветов.)
  • Отрицательное предложение с **any**: "They don’t have any money." (У них нет денег.)
  • Вопрос с **any**: "Is there any coffee left?" (Остался ли кофе?)

Практика:

Преобразование утвердительных предложений с **some** в вопросы и отрицательные предложения с **any**:

  • "We have some books." → "Do we have any books?" → "We don’t have any books."
  • "She bought some apples." → "Did she buy any apples?" → "She didn’t buy any apples."
