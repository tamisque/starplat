Единственное и множественное число существительных (-s, -es)

Множественное число существительных

Темы урока:

  • 1. Введение
  • 2. Основные правила образования множественного числа
  • 3. Исключения из правил
  • 4. Примеры и практика
  • 5. Домашнее задание

Введение:

  • Понимание разницы между единственным и множественным числом важно для правильного употребления существительных.
  • Знание правил образования множественного числа помогает избежать ошибок в речи и письме.

Основные правила образования множественного числа с помощью -s и -es:

  • Добавляем -s к большинству существительных:
    • Пример: cat → cats (кот → коты)
    • Пример: book → books (книга → книги)
  • Добавляем -es к существительным, оканчивающимся на -s, -sh, -ch, -x или -z:
    • Пример: bus → buses (автобус → автобусы)
    • Пример: dish → dishes (блюдо → блюда)
    • Пример: church → churches (церковь → церкви)
    • Пример: box → boxes (коробка → коробки)

Исключения из правил:

Некоторые существительные имеют неправильные формы множественного числа:

  • man → men (мужчина → мужчины)
  • woman → women (женщина → женщины)
  • child → children (ребенок → дети)
  • mouse → mice (мышь → мыши)
  • foot → feet (нога → ноги)

Примеры и практика:

Преобразуйте следующие существительные из единственного числа в множественное:

  • apple → apples (яблоко → яблоки)
  • lady → ladies (леди → леди)
  • party → parties (вечеринка → вечеринки)
  • city → cities (город → города)
  • baby → babies (младенец → младенцы)
