Конструкции There is и There are

There is / There are

Конструкции there is и there are используются для описания наличия объектов и их количества. Эти конструкции помогают создавать ясные предложения и описывать окружающий мир.

Темы урока:

  • 1. Принцип использования
  • 2. Примеры
  • 3. Когда использовать there is и there are
  • 4. Типы предложений
  • 5. Домашнее задание

Принцип использования:

  • There is используется для обозначения одного объекта (единственное число).
  • There are используется для обозначения нескольких объектов (множественное число).

Примеры:

  • There is a book on the table. (На столе есть книга.)
  • There are three apples in the basket. (В корзине три яблока.)

Когда использовать there is и there are:

There is: Используется с существительными в единственном числе.

  • There is a dog in the yard. (Во дворе есть собака.)
  • There is a car parked outside. (Снаружи припаркована машина.)

There are: Используется с существительными во множественном числе.

  • There are two dogs in the yard. (Во дворе две собаки.)
  • There are many books on the shelf. (На полке много книг.)

Типы предложений:

Утвердительные предложения:

Используются для утверждения о наличии объектов.

  • There is a student in the classroom. (В классе есть студент.)
  • There are five chairs in the room. (В комнате пять стульев.)

Отрицательные предложения:

Используются для отрицания наличия объектов. Для этого добавляется not после is или are.

  • There is not (isn’t) a cat in the garden. (В саду нет кота.)
  • There are not (aren’t) any cookies left. (Печенья не осталось.)

Вопросительные предложения:

Для формирования вопросов ставим is или are перед there.

  • Is there a restaurant nearby? (Рядом есть ресторан?)
  • Are there any questions? (Есть вопросы?)

Домашнее задание:

  • Напишите описание (5-7 предложений) своего идеального места или комнаты, используя конструкции there is и there are.
  • Создайте постер или презентацию с изображением этого места, подписав объекты с использованием there is и there are.
